Summary

Visio is a Visual Object Recognition platform. For giving its clients from retail industry such as bakeries, users can manage and train their menus and products and merge the trained menus into their POS devices through the services provided.

It is featured with device delegation toward stores, Auto Object Recognition Labelling from products, immediately or scheduled training sessions for menus, immediately or scheduled deployment for the trained menu.

Frontend JavaScript Library: ReactJS
Localization: i18n - "Lightweight simple translation module with dynamic JSON storage."
UI Framework: Material UI (v4)
Data Table: Material Table- "React data table component that is based on material-ui."
Repository Management: Sourcetree, Bitbucket
Remote Data: REST APIs
