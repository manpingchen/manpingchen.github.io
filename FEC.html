Summary

FEC Platform is a software web application. It allows authorized users to manage accounts and permissions, POS device delegation, and subscribe services.

It is featured with Account Management (CRUD), Company Management, Device Delegation, Subscription Management and Permission and Role Management.

Frontend JavaScript Library: ReactJS
Localization: i18n - "Lightweight simple translation module with dynamic JSON storage."
UI Framework: Material UI (v4)
Data Table: Datatable- Provided an advanced data table controls to our application
Repository Management: Sourcetree, Bitbucket
Remote Data: REST APIs
