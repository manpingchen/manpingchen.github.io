Summary

Nubis is a POS device management platform for its global clients, aiming to provide a consistent device control and power management.

It is featured with Utility Control, Device Health Status provided as real-time data, Alarm Notification, and immediately or scheduled multiple task to single device or grouped devices, such as Screenshot through VNC, File Deployment, Screen Rotation, etc.

Frontend JavaScript Library: ReactJS
Localization: i18n - "Lightweight simple translation module with dynamic JSON storage."
UI Framework: Material UI (v4)
Notification: Noty - "A Dependency-free notification library"
Data Visualisation: ChartJS - "Simple yet flexible JavaScript charting for designers & developers"
Data Table: Datatable- Provided an advanced data table controls to our application
Repository Management: Sourcetree, Bitbucket
Remote Data: REST APIs
